# Educa-Grana

**Projeto da Atividade Extensionista Uninter**

Aplicativo web de educação financeira para comunidades de baixa renda, ensinando habilidades financeiras básicas e fornecendo acesso a informações sobre programas de assistência social.

## Funcionalidades

- **Educação Financeira**: Módulos sobre orçamento pessoal, como poupar, crédito consciente e reserva de emergência
- **Programas de Assistência Social**: Informações sobre Bolsa Família, BPC/LOAS, Auxílio Gás, Tarifa Social de Energia, CadÚnico, Farmácia Popular, Minha Casa Minha Vida e cursos gratuitos
- **Calculadora de Orçamento**: Ferramenta interativa para planejar receitas e despesas mensais com dicas personalizadas

## Tecnologias

- [React](https://react.dev/) + [Vite](https://vite.dev/)
- [React Router](https://reactrouter.com/) para navegação
- CSS puro com variáveis e design responsivo

## Como executar

```bash
npm install
npm run dev
```

Acesse em http://localhost:5173

## Como construir para produção

```bash
npm run build
```
